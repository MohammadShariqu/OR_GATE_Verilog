# OR Gate in Verilog

## Overview

This project implements a **2-input OR Gate** using **Verilog HDL** along with a testbench for simulation and verification. The design is compatible with **EDA Playground** and generates waveforms that can be viewed using **EPWave**.

## Project Files

* `or_gate.v` – Verilog implementation of the 2-input OR gate.
* `or_gate_tb.v` – Testbench for simulating and verifying the OR gate.
* `waveform.png` – Output waveform generated during simulation (optional).

## Features

* Simple and easy-to-understand Verilog code.
* Complete testbench covering all input combinations.
* Compatible with EDA Playground.
* Supports EPWave for waveform visualization.
* Suitable for beginners learning digital logic design.

## Simulation

The testbench applies all possible input combinations to the OR gate and verifies the output. A VCD file is generated, allowing the waveform to be analyzed using EPWave.
